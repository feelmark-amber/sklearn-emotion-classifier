# 指示 (Instruction)

あなたは、様々な文学作品（小説、詩、エッセイなど、ジャンルは問わない）を読んだ多様な読者をシミュレートするAIです。
これから、AIの学習データセットを作成するために、人間が書いたような「生々しい」感想テキストを複数生成してください。

**【今回の特別な指示】**
* **独自性と多様性の重視:** これまでに生成した可能性のある感想とは**異なる視点、異なる感情の組み合わせ、異なる言い回し**を積極的に用いて、**毎回ユニークな**感想セットになるように強く意識してください。ありきたりな表現（例：「鳥肌が立った」「一気読みした」「考えさせられた」など）はなるべく避け、より具体的で個人的な反応をシミュレートしてください。
* **多様な作品への反応:** ファンタジー、SF、ミステリー、恋愛、純文学、ホラー、エッセイなど、**異なるタイプの作品を読んだかのような**、多様な感情反応（感動、批判、考察、共感、困惑、興奮、静かな余韻など）が自然に混ざるように生成してください。特に、**少しひねくれた感想や、予想外のポイントに言及する感想**なども含めてみてください。

# 生成する感想テキストの要件 (基本)

* **文体・内容:**
    * 人間が書きそうな、自然で「生々しい」感想にしてください。
    * 口語表現、多少の砕けた表現、感情の直接的・間接的な表現（例: 「泣いた」「最高」「ヤバい」「ｗ」「！！」「…」）を含めてください。
    * ポジティブ、ネガティブ、両方の混在、感動、批判、混乱、考察、キャラクターへの言及など、**幅広い**反応を含めてください。
    * 様々な長さ（短い一文から、少し長めの段落まで）のテキストを生成してください。
    * 特定の作品名やキャラクター名は「あの場面」「主人公」「この作品」、`〇〇` や `△△` 等で表現してください。
    * 完璧に整っておらず、少し話が飛んだり主語が抜けたりするような自然なニュアンスも出してください（意味不明にならない程度に）。
* **量:** **15個** の異なる感想テキストを生成してください。
* **禁止事項:** 実在レビューのコピー禁止。ヘイトスピーチ等不適切内容禁止。全て架空の感想とすること。**以前の指示で生成したものと酷似した内容にならないように注意してください。**

# 出力フォーマット要件

* 生成した各感想テキストを、**JSON形式**で出力してください。全体を `[` と `]` で囲んだリスト形式とします。
* リストの各要素は、**`"id"` と `"data"` という２つのキーを持つJSONオブジェクト**です。
* `"id"` には数値型の連番を **（【★ここに前回の最後のIDの次の番号を指定してください★】例: 106）** から順に振ってください。
* `"data"` の値は、 **`"text"` というキーのみを持つオブジェクト**とし、その値を生成した感想テキスト（文字列）にしてください。
* **各JSONオブジェクトはカンマ `,` で区切ってください。** リストの最後の要素の後にはカンマを付けないでください。

# 出力例 (JSON形式) - 形式参考用

```json
[
  { "id": 106, "data":{"text": "(ここに生成された感想テキスト)" }},
  { "id": 107, "data":{"text": "(ここに生成された感想テキスト)" }}
  // ... 以下、合計15個分 ...
]
```

上記の形式に従い、**毎回異なる、多様で生々しい感想テキスト**を15個生成してください。**(※IDはこちらで指定しますので、【★開始ID★】から始めてください)**
